Elements Definitions

Heading Elements

<h1>
This is the Main Heading.It is the larget size possible
<h2>
This is a level 2 heading. It is used for sub headings
<h3>
This is the level 3 heading and can used for subsections under level 2
<h4>
This is the level 4 heading and can used for sections under level 3
<h5>
This is a level 5 heading and can be used for sections under level 4
<h6>
This is a level 6 heading.it is the smallest of all the heading and can be used for sections under level 5

Paragraphs

<p>
The paragraph element is used to start or create a new paragraph.

Bold And Italics

<b>
This is used to make a text bold or more noticeable than other text
<i>
This is the tag used to make text appear in italics

Superscript and Subscript

<sup>
The element is used to contain characters that are superscripts.An example includes July 4th
<sub>
This element is used to contain characters that are subscripts.An example inclues H2O

Line Breaks and Horizontal Rules

<br/>
This element is used to break a line
and begin a new one
<hr/>
This element is used to create a break between themes or topics

Strong and Emphasis

<strong>
This element is used to show importanceor strong emphasis
<em>
This element indicates emphasis and is used to subtly change the meaning of a sentence

Quotations

<blockquote>
This element is used for long quotes that can take an entire paragraph. The paragraph element is used inside the blockquote

Rain, Rain , go away. Come again another day. Little Children want to play

<q>
This element is used for quotations

Example:Well! Well! Well! Isn't it a good day! said Harry the Pig

Abbreviations And Acronyms
<abbr>
This element is used for both abbreviations and acronyms in HTML

Dr.Jones is the best plastic surgeon in the country

NBA is the Basketball regulatory body in the USA

Citations And Definitions

<cite>
This is element is used when referencing a piece of work

Macbeth by Shakespeare is a very tragic play

<dfn>
This element is used to define a term

A line is the shortest distance between 2 points

Author Details

This element is used to contain the contact information of the author of the page
